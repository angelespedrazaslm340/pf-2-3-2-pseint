# FizzBuzz

## Requisitos

- PSeint

## Instalación

1. Descargar o clonar el repositorio

## Instrucciones

Escribir un programa con las siguientes características, usando pseudocódigo.

1. El programa debe imprimir los números del 1 al 100.
2. Una vez que tu programa hago esto hacer los siguientes puntos.
3. Remplazar los números que sean múltiplos de 3 por la palabra Fizz.
4. Remplazar los números que sean múltiplos de 5 por la palabra Buzz.
5. Remplazar los números que sean múltiplos de 3 y 5 por la palabra FizzBuzz.



Ejemplo de Salida

1

2

Fizz

4

Buzz

Fizz

7

8

Fizz

Buzz

11

Fizz

13

14

FizzBuzz

## Entrega
Para la entrega de este repositorio necesitaras una captura de tu pseudocódigo
y del diagrama de flujo que genera PSeint, en caso de que tengas error para mostrar el diagrama 
anotarlo en la entrega del laboratorio.
